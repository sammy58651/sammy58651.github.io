# p2 小Y與小T的積木遊戲(Small Y and Small T's Bricks Game)

**思路**

任意一個數字拆到最後都會是 1，所以就會有 n 個 1。所以一個數字拆成兩個的話，就會增加 1 個數字，直到全部都是 1 為止

Time : $O(N)$
Code:
```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    int n;
    int s,t;
    cin>>n;
    vector<int>a(n);
    for(int i = 0;i<n;i++){
        int b;
        cin>>b;
        a[i] = b;
    }
    s = accumulate(a.begin(),a.end(),0);
    t = s - n;
    if (t%2==0){
        cout<<"Small T wins!!!";
    }
    else{
        cout<<"Small Y wins!!!";
    }
}

```
